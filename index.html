What is this?

This is a test project created to practice deploying a static website to Vercel. It includes a homepage, an About Us page, and some styled components to get you started.

Feel free to modify the content, swap out colors, or extend it with JavaScript functionality. The goal is to get comfortable with the deploy flow.
